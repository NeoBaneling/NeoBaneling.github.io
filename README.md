## Commands

### `yarn dev`

Starts dev server on [localhost:4321](http://localhost:4321).

### `yarn build`

Builds the app.

### `yarn preview`

Runs the build.

### `yarn fmt`

Runs prettier formatter.

### `yarn fmt:fix`

Runs prettier formatter with auto-writing enabled.

### `yarn lint`

Runs eslint checks.

### `yarn lint:fix`

Runs eslint checks with auto-fixing enabled.
